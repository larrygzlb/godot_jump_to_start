# godot_jump_to_start

Minimal Godot learning starter project.

## What is initialized
- Godot project configuration (`project.godot`)
- Blank main scene (`scenes/Main.tscn`)
- Basic project icon (`icon.svg`)
- Godot-focused `.gitignore`
- Empty folders for organization

## Suggested learning flow
1. Open the project in Godot.
2. Inspect the node tree in `Main.tscn`.
3. Create your own test scenes under `scenes/`.
4. Add scripts later when you are ready to start logic.

## Folder layout
- `scenes/` for `.tscn` files
- `scripts/` for future `.gd` files
- `assets/` for textures, audio, and fonts


Required Godot version: 4.6.stable
