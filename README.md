# godot_jump_to_start

Godot learning project with small practice games.

## Current games
- `game_0`: Platformer starter.
- `game_1`: Move 5 colored items to matching target positions.

## Run a game
1. Open the project in Godot.
2. To run `game_0`, open `res://game_0/scenes/Main.tscn` and press Play Scene.
3. To run `game_1`, open `res://game_1/scenes/Main.tscn` and press Play Scene.

## game_1 controls
- Left click: Select an item.
- Arrow keys or WASD: Move selected item.
- Win condition: Place all 5 items on their matching targets.
- After winning: Choose Restart or End.

Required Godot version: 4.6.stable
